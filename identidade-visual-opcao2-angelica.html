03 · Textura & Humanização

O que tira o post
da cara de catálogo.

Textura é o que faz um post parecer feito por uma pessoa, não gerado por uma máquina. Aplicada com sutileza, ela adiciona profundidade e calor sem chamar atenção para si mesma.

✦

Regra da textura: o grain nunca deve ser visível como "efeito". Se der para perceber, está forte demais. A função é subconsciente — o olho sente, não vê. Opacidade ideal: 4–6%.

Como aplicar grain no Canva: Baixe uma imagem de textura grain PNG com fundo transparente. No Canva, adicione como elemento acima de tudo, ajuste a transparência para 5–8% e mude o modo de mesclagem para Multiplicar (fundos claros) ou Sobrepor (fundos escuros). Busque "grain texture transparent PNG" no Google Images.

04 · Grid do Perfil

O ritmo visual
que cria reconhecimento.

O grid não é uma grade rígida — é um ritmo. Escuro, claro, escuro. Quem visita o perfil sente a consistência antes de ler uma palavra.

📐

Padrão do grid: A cada 9 posts, o ciclo se repete em blocos de 3×3 coesos. Nunca 2 claros seguidos. Nunca 3 escuros sem 1 claro entre eles. O ritmo correto: escuro · claro · escuro / escuro · escuro · claro / claro · escuro · escuro.

07 · Tutorial Canva

Como montar
cada template do zero.

Passo a passo para criar os templates base e replicar com consistência em todos os posts.

Configuração inicial — fazer uma vez só

1
Criar o Kit de Marca no Canva Pro

Canva → Marca → Criar Kit de Marca. Adicione as cores: #7C3E52 #6A3346 #5A2A3A #D4B896 #EAD5BF #F2E8EC #E8D5DD #6B7C6A #FAF7F2 #2A1F24. Nomeie cada uma conforme a paleta.

2
Adicionar as 3 fontes ao Kit

No Kit de Marca, adicione: Cormorant Garamond (título), Jost (corpo) e DM Mono (handles). Todas gratuitas no Canva. Configure: título Cormorant Bold 36px · subtítulo Cormorant Light Italic 28px · corpo Jost Light 14px.

3
Criar a pasta "AP — Templates"

No Canva, crie uma pasta com esse nome. Todos os templates base ficam aqui. Para cada post novo: abra o template → clique em "Duplicar design" → edite a cópia. Nunca edite o original.

Criando a capa de carrossel escura (template base)

1
Novo design 1080×1080px

Canva → Criar design → Tamanho personalizado → 1080 × 1080 px. Nomeie "Template — Capa Escura".

2
Fundo rosa vinho com textura

Fundo: #7C3E52. Adicione o PNG de grain acima do fundo, transparência 5%, modo de mesclagem Sobrepor. Isso dá profundidade imediata.

3
Ornamento linha-losango-linha

Linha horizontal 2px, cor #D4B896, opacidade 50%. Quadrado girado 45° (8×8px, mesma cor) no centro. Agrupe os 3 elementos. Posicione a 140px da base à esquerda.

4
Título principal

Fonte: Cormorant Garamond Bold, 72px, branco. Alinhado à esquerda, margem esquerda 64px. Itálicos em Cormorant Light Italic, cor #EAD5BF.

5
Handle e numeração

Handle: DM Mono Regular, 20px, #D4B896, opacidade 50%. Posicione na base esquerda. Numeração (01/07): DM Mono 18px, mesma cor, opacidade 40%. Canto superior direito.

6
Círculos decorativos (opcional)

Círculos vazados, borda 1px, #D4B896, opacidade 12%. Posicione parcialmente fora do canvas nos cantos — criam profundidade sem poluir.

7
Salvar como template base

Salve na pasta "AP — Templates". Para cada post: Arquivo → Fazer uma cópia. Edite apenas o texto — nunca altere cores ou fontes.

💡

Para o slide claro (bege rosado): fundo #F2E8EC, grain em modo Multiplicar 3%, borda esquerda 3px em #7C3E52, título Cormorant Garamond Bold cor #2A1F24, tag em Jost SemiBold 12px all caps cor #7C3E52.

08 · Regras de Uso

O que sempre fazer.
O que nunca fazer.

✅
Sempre use as cores da paleta

Nenhuma cor fora da paleta entra no feed. Para variações, use opacidades das cores existentes.

✅
Mantenha o ritmo escuro/claro

A cada 3 posts: no mínimo 1 claro e 1 escuro. O ritmo é o que cria identidade visual no grid.

✅
Grain em todos os posts

Mesmo sutil, aplique sempre. É o elemento que unifica todos os posts e tira a aparência de template genérico.

✅
Ornamento em toda capa escura

A sequência linha-losango-linha em champagne é o "selo" visual. Toda capa escura tem esse ornamento.

❌
Nunca misture mais de 2 fontes

Três fontes em um mesmo post destroem a hierarquia visual e o post perde elegância.

❌
Nunca use fundo branco puro

O branco #FFFFFF não existe na identidade. Use marfim #FAF7F2 ou bege rosado #F2E8EC.

❌
Nunca poste 2 claros seguidos

No grid, dois posts claros lado a lado quebram o ritmo e apagam a identidade visual.

❌
Nunca reduza o handle abaixo de 10px

O @angelicapires_s precisa ser legível mesmo em miniatura. Tamanho mínimo no Canva: 20px.

💛

Resumo em uma frase: Rosa vinho + grain + Cormorant Garamond + ornamento = Angélica Pires. Toda vez que esses 4 elementos aparecem juntos, o post é imediatamente reconhecível — mesmo sem ler o handle.
